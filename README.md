# electron-my-computer
基于electron，模仿win10上的&lt;我的电脑>
### 目标：

尽可能还原常见功能，同时学习使用electron和nodejs相关技术点



### 计划功能：

- [x] win10我的电脑布局
- [x] 最小化/最大化/关闭按钮
- [x] 双击窗口标题栏最大化
- [ ] 渲染左侧目录结构
- [ ] 渲染右侧展示列表
- [ ] 双击进入文件夹
- [ ] 展示当前路径
- [ ] 统计当前项目总数



### 启动项目：

`npm i`

`npm run start`



### 版本依赖：

`node`：12.18.3

`electron`：10.1.5

`bootstrap`：3.4.1

`jquery`：1.9.1

`electron-reload`：1.5.0
